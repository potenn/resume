김민기

## 자기 소개

- iOS 어플리케이션 개발에 자신이 있습니다.
- Node.js의 Express Framework를 이용하여 서버 / 클라이언트 어플리케이션을 만들 수 있습니다.
- 기술을 학습하고 공유하는 것을 좋아합니다.
- 알고리즘 공부하는 것을 좋아합니다. (LeetCode와 백준 온라인 저지에서 주로 풉니다.)
